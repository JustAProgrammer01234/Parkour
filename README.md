# Parkour
A basic parkour game made using Unity. 

## Controls
`Esc` **-** Pause
 
`W` **-** Move forward

`A` **-** Move left

`S` **-** Move backward

`D` **-** Move right
 
`Space Bar` **-** Jump

`Shift` **-** Crouch
